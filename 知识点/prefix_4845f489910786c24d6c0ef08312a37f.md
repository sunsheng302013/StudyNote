**简要描述：**

-获取已创建的网络名

**请求URL：**
- `/snac/deploy/v1.0/network/names`

**请求方式：**
- GET

 **返回参数说明**

|参数名|类型|说明|备注|
|:-----  |:-----|:-----|:-----|
|networkName|String|已经创建的网络的名字|||

 **返回示例**

 ```
 {
  "code": "SUCCESS",
  "message": "operate success",
  "data": [
    "net1"
  ]
}

 ```



 **备注**

- 更多返回错误代码请看首页的错误代码描述
